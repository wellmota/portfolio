# Logos dos clientes (logo wall)

Solte aqui os arquivos de logo e eles aparecem **automaticamente** no carrossel,
substituindo o wordmark de texto. Enquanto o arquivo não existir, mostra o nome
em texto (fallback).

## Como usar

1. Coloque o arquivo com o **nome exato** abaixo (de preferência **`.svg`**;
   `.png` com fundo transparente também funciona).
2. Pode ser o logo em qualquer cor — o CSS normaliza tudo para **branco
   monocromático** (fica uniforme no fundo escuro) e acende no hover.
